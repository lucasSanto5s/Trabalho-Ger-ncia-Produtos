Projeto de Implementação - Loja de Suplementos


- Descrição
Este é um projeto de implementação para uma loja de suplementos, desenvolvido como parte da disciplina de programação. O sistema visa gerenciar o estoque e as vendas de uma loja de suplementos.


- Funcionalidades
Login e autenticação de usuários (gerentes de estoque).
Cadastro e gerenciamento de produtos(Adição, Edição e Remoção);
Controle de estoque;
Realização de vendas;


- Tecnologias Utilizadas
Java (linguagem de programação);
NetBeans (IDE de desenvolvimento);
MySQL (banco de dados);
Maven (gerenciador de dependências);
Swing (framework para criação de interfaces gráficas).


- Requisitos para Instalação
Java JDK 8 ou superior instalado;
IDE NetBeans instalada;
Servidor MySQL instalado e em execução;
Conexão com a internet para download de dependências do Maven.


- Como Instalar
Baixe o repositório do projeto;
Importe o projeto para a IDE NetBeans;
Configure a conexão com o banco de dados MySQL: Abra o MySQL -> crie uma conexão -> clique em File -> Open SQL Script -> selecione o arquivo disponibilizado (SuplementosScript) -> após abrir o arquivo dentro do MySQL, execute todas as linhas. Agora seu banco de dados está em execução;
Execute o arquivo "Main.java" para iniciar o aplicativo;
